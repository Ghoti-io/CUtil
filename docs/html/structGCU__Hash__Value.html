Detailed Description

Container used to return the result of looking for a hash in the hash table.

Although it may seem strange to return a value as part of a structure, especially when the programmer undoubtedly just wants the value, it is also imperitive that the hash table be able to indicate whether or not the value existed in the table. Both goals are accomplished by this approach.
